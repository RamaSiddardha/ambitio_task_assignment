# Frontend Developer Intern Assignment

## Hero Section Implementation

This project is a hero section developed for the landing page as per the provided Figma design. The implementation ensures pixel-perfect accuracy, responsiveness, and smooth animations to enhance the user experience.

## Tech Stack

- **Next.js / React.js** – For building the UI components
- **Tailwind CSS** – For styling and responsiveness
- **Framer Motion** – For animations and interactions

## Features

- **Pixel-perfect design**: The layout, typography, colors, and spacing match the Figma design.
- **Responsive UI**: The hero section adapts seamlessly to desktop, tablet, and mobile screens.
- **Smooth animations**: Added creative animations using Framer Motion to improve interactivity.

## Setup & Installation

1. Clone the repository:
   ```sh
   git clone [repository-link]
   ```
2. Navigate to the project folder:
   ```sh
   cd project-folder
   ```
3. Install dependencies:
   ```sh
   npm install  # or yarn install
   ```
4. Run the development server:
   ```sh
   npm run dev  # or yarn dev
   ```
5. Open `http://localhost:3000/` in your browser to view the hero section.

## Deployment

The project can be deployed using **Vercel** or **Netlify** for a live preview.
